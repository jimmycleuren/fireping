# Short Description
An alert destination is used do define where to store alerts.

# Required fields
- `Name` of the alert destination
- `type`, where the alerts go to 
    - option `http`: broken
    - option `mail`: broken
    - option `monolog`: Will log the alert in the log files
    - option `slack`: broken
